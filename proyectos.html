Proyectos

X-STAR

Este proyecto es un sitio web de una tienda de ropa llamada X-STAR, dentro de la misma se pueden encontrar la página principal con la Nueva Colección, la página con los Productos y otra página con un formulario de Contacto.
Para su realización se utilizaron conocimientos de HTML como tablas, listas desordenadas y formularios, junto con un CSS externo anclado a todas las páginas.

X-STAR

Coffee-blog

Este proyecto es la página principal de un Blog de Cafeterías de la ciudad de Mar del Plata, en donde se brinda información detallada de las mismas y más adelante podría llegar a ser un sitio web con un conjunto de páginas web enlazadas en donde cada una permitiría la descripción completa de cada cafetería.
Para su realización se utilizó en su mayor parte Bootstrap con Grids y una barra de navegación. El CSS es interno ya que por el momento es una sola página HTML.

Coffee-blog

My gallery

Este proyecto intenta recrear una galería virtual, en este caso de obras renacentistas y capillas con fantásticas estructuras y vitrales.
Para su realización se utilizó en su mayor parte Bootstrap y un CSS interno ya que es una sola página HTML.

My gallery

Byas

Este proyecto llamado Byas es el sitio web de una Agencia de Viajes con el viaje a Nueva York como principal atractivo, además tiene una página con todos los Destinos que ofrece y otra con un formulario de Contacto para poder consultar por cualquiera de sus ofertas.
Para su realización se utilizaron conocimientos de HTML junto con un CSS externo anclado a todas las páginas.

Byas

Gazelle Lo Pro

Este proyecto es una página web creada para un Producto Nuevo, en este caso las zapatillas Gazelle Lo Pro, está compuesta por varios elementos audiovisuales y texto informativo.
Para su realización se utilizó un carrusel de Bootstrap, un video sacado de Tiktok, un audio en loop y un CSS interno.

Gazelle Lo Pro

Howl's Moving Castle

Este proyecto es una página web sobre una película de Studio Ghibli llamada El Castillo Ambulante, que contiene una breve descripción de la historia, los personajes y un análisis de sus temáticas.
Para su realización se utilizaron conocimientos de HTML como listas desordenadas, formularios y enlaces a otras páginas. El CSS figura de manera interna con aparición de algunos floats.

Howl's Moving Castle
